# Instalação:

**Pré Requisitos:** java e maven

```shell
git clone https://github.com/lima-anderson/consultorio-api.git
```
```shell
cd consultorio-api/
```
```shell
mvn install
```
```shell
cd target/
```
```shell
java -jar consultorio-api-0.0.1-SNAPSHOT.jar
```
